# Green Lava
Green Lava is an Debian based Minecraft Server hosting OS. It is meant to have an Minecraft server running out of the box.  

## How to contribute
Run the `build.sh` script to get your own os image.  
It might need to be made executable with `chomod +x build.sh`  
Make changes in the build.sh script to make changes to the project
Making changes to the ./config direcory is not recommended, you should instead add parameters to the build.sh 



This OS may bundle third-party packages under their respective licenses.

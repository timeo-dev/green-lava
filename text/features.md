# FEATURES

The features that are planned for Green Lava include:

1. Minecraft server
| An Minecraft server in the version that the user desires. Installed from an script called `install-server.sh`

2. Modded servers
| Modded servers in the form of fabric mods will be installed from `install-server.sh` too.

3. SSH
| SSH will be installed if the user agrees to it in the installation process

3. Default packages
| Green Lava will contain some default packages which will be listed here. More packages can be installed with apt.
    * fastfetch
    * curl
    * wget
    * vim
    * unzip
